# Product Overview

This product is a 7-day sprint-based personal operating system.

It is NOT:
- A habit tracker
- A motivation app
- A gamified productivity tool

It IS:
- A structured execution system
- A decision-fatigue reducer
- A weekly optimization engine

---

# Core Promise

Fewer decisions.  
Clear execution.  
Weekly optimization.

---

# Product Philosophy

1. No daily negotiation.
2. Rules are locked during a sprint.
3. Conscious weekly revision.
4. AI appears only at structured decision moments.
5. Minimal surface area.

---

# Sprint System

## Sprint Structure

- Default sprint duration: 7 days
- PRO can unlock 14-day sprints
- First sprint is always 7 days for every user
- Max 3 rules per sprint
- Min 1 rule per sprint
- Rules are single-line (binary or numeric)

---

## Day 1 Calibration

Only at the end of Day 1:
- Add rule allowed
- Drop rule allowed
- Adjust targets allowed

From Day 2 onward:
- Rules are locked

Purpose: sustainability first, discipline after.

---

## Active Sprints

FREE:
- 1 active sprint

PRO:
- Up to 3 active sprints
- Optional categorization (Health, Work, Mind, etc.)

Categories are organizational only. No heavy analytics.

---

## Daily Flow

- Day X / Y indicator
- Rule checklist
- Minimal completion feedback
- No gamification
- No progress graphs

---

## Daily One-Line Entry

Prompt: "One line."

- No edit
- No delete
- Displayed collectively at sprint end

Purpose: clarity record, not journaling.

---

# Streak System

Daily streak:
- Simple text only
- No gamification visuals

Sprint streak:
- Count of completed sprints
- Long-term consistency emphasis

---

# AI System (PRO Only)

AI is:
- Time-based
- Structured
- Not always-on
- Not unlimited chat

AI provider is abstracted behind a gateway and can be switched without mobile changes.

---

## 1. Story Session

- 45–60 minutes
- No message cap
- Time-limited

Output:
- Operational Profile
- Sprint Design Recommendations
- Tone Preferences

Only structured summary is stored.
Raw transcript is not stored by default.

---

## 2. Sprint Planning Session

- 5–8 minutes
- Sustainability and balance check
- Time-limited

---

## 3. Day 1 Calibration AI

- 5 minutes
- Optimize rule set
- Time-limited

---

## 4. Sprint End Review

- 8–10 minutes
- Performance analysis
- Optimization suggestions
- Time-limited

---

## 5. Weekly AI Meetings

- 3 per week
- 10–15 minutes each
- No message cap
- Time-limited

AI acts as a structured decision assistant, not a therapy chatbot.

---

# Free vs Pro

## FREE

- 1 active sprint
- Max 3 rules per sprint
- Fixed 7-day duration
- Last 3 sprint history
- Daily one-line entry
- Basic streak
- No AI
- No persistent profile

## PRO

- 3 active sprints
- 14-day sprint option
- Unlimited sprint history
- Sprint comparison view
- Story Session
- All AI modules
- Weekly AI meetings
- Persistent operational profile
- Context Vault management
